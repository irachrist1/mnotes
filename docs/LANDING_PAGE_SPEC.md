## Animations & Micro-interactions

### Page Load
- Hero content fades up with 0.4s delay stagger
- Background gradient slowly animates (subtle)
- Dashboard preview slides up from bottom

### Scroll Animations
- Sections fade in + slide up when entering viewport
- Use Intersection Observer with 0.1 threshold
- Stagger child elements by 0.1s

### Interactive Elements
- **Buttons:** Scale to 1.02 on hover, slight shadow increase
- **Cards:** Lift 4px on hover with shadow expansion
- **Links:** Underline slides in from left
- **Dark mode toggle:** Smooth 0.3s color transitions throughout

### Dashboard Preview
- Subtle "live" data animation (numbers occasionally updating)
- Cursor animation showing interaction
- Optional: Play/pause for accessibility

### Waitlist Form
- Success state with confetti or checkmark animation
- Error state with gentle shake
- Loading state with spinner

## Copywriting Guidelines

### Voice & Tone
- **Direct:** No fluff. Say what it does.
- **Personal:** Use "you" and "your" frequently
- **Confident:** We're building something valuable
- **Honest:** It's early stage â€” own that authentically

### Words to Use
- Context, intelligence, insights, clarity
- Capture, track, understand, grow
- Personal, yours, you
- Finally, actually, real

### Words to Avoid
- Revolutionary, game-changing, disruptive
- Synergy, leverage, optimize (overused)
- "We're excited to announce" (corporate speak)
- Fake social proof or user counts

### Headline Formula
**[Benefit] + [Differentiator]**
- "Finally, an AI That Actually Knows You"
- "Your Ideas Deserve Better Than a Notes App"
- "Stop Explaining Your Life to AI"

## Accessibility Checklist

- [ ] All images have alt text
- [ ] Color contrast meets WCAG AA (4.5:1 for text)
- [ ] Interactive elements have focus states
- [ ] Animations respect `prefers-reduced-motion`
